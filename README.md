# AmritaHub 🎓

> Collaborative study resource platform for Amrita School of Engineering — CSE

![React](https://img.shields.io/badge/React-18-61DAFB?logo=react)
![Firebase](https://img.shields.io/badge/Firebase-10-FFCA28?logo=firebase)
![License](https://img.shields.io/badge/license-MIT-green)

AmritaHub is a student-built web platform where Amrita CSE students can 
discover, share, and organise study materials — notes, PDFs, video lectures, 
articles and practice sets — all mapped to the official B.Tech CSE 2023 curriculum.

Built with React + Firebase. No ads. No gatekeeping. Just students helping students.
